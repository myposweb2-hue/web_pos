Quick Integration Guide

1

View Templates

Open templates in browser to preview the professional invoice designs. Navigate to:
/invoices/preview/a4 /invoices/preview/a5 /invoices/preview/receipt
2

Print to Your Printer

Use browser print dialog (Ctrl+P or Cmd+P) to configure and print:
Margins: 0.5" (A4/A5) or 0" (Thermal) Scale: 100% (no scaling) Paper: A4, A5, or Thermal 80mm Color Mode: CMYK or Color
3

Integrate with Python

Use the invoice generator in your Flask routes:
from app.utils.invoice_generator import InvoiceGenerator generator = InvoiceGenerator() html = generator.generate_a4_invoice(invoice_data)
4

Use REST API

Generate invoices via POST request:
POST /invoices/api/generate Content-Type: application/json { "format": "a4", "invoice_data": { ... } }
5

Customize & Deploy

Personalize templates with your branding and deploy to production:
  • Edit company logo in HTML
  • Update brand colors in CSS
  • Modify terms and conditions
  • Test with all printer types
  • Deploy to production server
