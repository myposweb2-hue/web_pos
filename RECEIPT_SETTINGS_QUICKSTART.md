# Quick Start: Dynamic Receipt Settings

## Summary

✅ **Receipt settings now dynamically update on printed receipts!**

When you update receipt settings in **Settings > Receipt Configuration** or **Settings > Terminal Configuration**, those changes automatically appear on the next receipt print.

## How to Use

### Step 1: Update Receipt Settings
1. Go to **Settings** (top navigation)
2. Click the **Receipt Configuration** tab
3. Update any of these fields:
   - Business Name
   - Business Address
   - Business Phone/Email/GST
   - Thank You Message
   - Warranty Information
   - Footer Text
   - Default Receipt Format
4. Click **"Save Receipt Settings"**
5. You'll see a success notification

### Step 2: Update Terminal Settings (Optional)
1. Click the **Terminal Configuration** tab
2. Update:
   - Receipt Type (Thermal/A4/A5)
   - Receipt Theme
   - Print Options (Auto-print, Show Logo, etc.)
   - Paper Size & Font Size
3. Click **"Save Terminal Settings"**

### Step 3: Print Receipt
1. Go to **Sales** module
2. Complete a sale or open an existing one
3. Click **Print** or Print Button
4. Your receipt will automatically use the updated settings!

## What Gets Updated

When you change settings, the following appear on receipts:

| Setting | Shows Up On Receipt | Example |
|---------|-------------------|---------|
| Business Name | Header (top) | "My Store" |
| Business Address | Under business name | "123 Main Street" |
| Business Phone | Contact section | "(555) 123-4567" |
| Thank You Message | Bottom of receipt | "Thank you for your business!" |
| Warranty Info | Bottom section | "1 year warranty" |
| Footer Text | Very bottom | "Generated by Web POS" |
| Show QR Code | Top/side | QR code appears/hides |
| Format | Template type | 80mm thermal, A4, or A5 |

## Troubleshooting

### Settings not appearing?

1. **Check if settings saved**
   - Look for green "Settings saved successfully!" notification
   - Try saving again

2. **Check receipt format**
   - Verify you selected the right receipt format (usually "Thermal 80mm")
   - Try different format to test

3. **Try preview**
   - Go to `/invoices/preview/receipt` in your browser
   - Should show current settings
   - If not showing, settings not saving

4. **Clear browser cache**
   - Press Ctrl+Shift+Delete
   - Clear cache and try again

### Logo not showing?

- Upload a small logo (200x100 pixels recommended)
- Supported formats: JPG, PNG, GIF
- File size: Under 1MB

### Settings changed back?

- Always click "Save" button after making changes
- Look for success notification
- Don't refresh page before saving

## Receipt Format Options

1. **Thermal (80mm)** - Standard POS receipt printer
   - Default: 80mm width
   - Best for: Quick sales, fast checkout
   
2. **A4** - Full page professional invoice
   - Format: 210mm × 297mm
   - Best for: Formal invoices, multi-item orders
   
3. **A5** - Half-page compact
   - Format: 148mm × 210mm
   - Best for: Condensed receipts, 2 per page

## Database Query (Advanced)

To verify settings are saved in database:

```sql
SELECT setting_key, setting_value FROM setting 
WHERE setting_category='receipt' 
ORDER BY setting_key;
```

You should see entries like:
- `business_name` → "Your Store"
- `thank_you_message` → "Thank you..."
- `footer_text` → "..."

## API Integration (For Developers)

### Get Current Settings
```bash
GET /api/settings/categories/receipt
```

### Save Settings
```bash
POST /api/settings
Content-Type: application/json

{
  "receipt": {
    "business_name": "New Store Name",
    "thank_you_message": "New message",
    "footer_text": "New footer"
  }
}
```
